# Pricing_Analysis_Project_CoffeGrind
The project is about a Coffee Grind Company, which is being negatively affected by the increasing COGS, and requires investigation and analysis into the products which require price adjustment or removal.
There are 5 data files: Sales data for 3 years, Product information, and Customer Information
SQL is used to clean the data and combine it into a single database so it can be used for further visualisation.
PowerBI is used to make a dashboard and present the findings
